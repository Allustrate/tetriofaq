性能提升

相较于其他网页游戏，TETR.IO 力求带给玩家完美的客户端游戏体验。若使用配置较低的设备，使用默认设置游玩时可能会出现卡顿。别担心！本条 FAQ 将给出一些解决卡顿的方法，按对游戏体验的影响程度从小到大排列。

1. 使用 TETR.IO 桌面版

TETR.IO 的桌面版针对性能做出了大幅优化，旨在为玩家提供更优的游戏体验，点击该链接下载。

2. 换个浏览器

Chrome 和 Chromium 内核的浏览器通常能比其它浏览器更流畅地运行游戏。如果你使用的是 Firefox 等其他浏览器，不妨试试 Chrome，游戏的流畅度将显著提升。（或者直接使用桌面版！）

3. 关闭节电模式

为保证续航、减少耗电量，大多笔记本默认开启节电模式。检查是否将电源策略调为高性能模式，以保证游戏流畅运行。

4. 确保浏览器/TETR.IO 桌面版使用高性能图形处理器

如果你的设备载有多块图形处理器（例如一块集成处理器 + 一块 NVIDIA 处理器），请确保浏览器或 TETR.IO 桌面版使用的是性能较高的那一块。为程序指定处理器的方式依设备而定，可前往 NVIDIA 控制面板（NVIDIA Control Panel）或 AMD Catalyst。在 Windows 10 及以上的操作系统中，也可通过系统的设置项指定处理器。

5. 在游戏中更改画面设置

这可能是最直接的一种方式，但也不妨查查试试。下面给出几个较明显的设置：

  • GRAPHICS — 控制画面的优劣。调至 MINIMAL 以获得最佳体验。 该选项将把一些子选项锁定至最佳值，这些选项在下方以星号（*）标记。
  • RENDER AT A LOW RESOLUTION* — 控制游戏是否以低分辨率运行，勾选后游戏画面更模糊，但大幅减少了处理器的运算量。 调至 ENABLE 以获得最佳体验。
  • WEBGL MODE — 游戏选用的 WebGL 版本，若游戏中出现异常粒子或闪光，请尝试更低的版本。调至 WEBGL 2 以获得最佳体验。
  • ACTION TEXT* — 控制上屏的动作文本。虽然动作文本会占用大量处理器资源，但有些动作文本较为重要，禁用前请三思。调至 OFF 以获得最佳体验。
  • PARTICLE COUNT* — 控制上屏的粒子数量。调至 10% 以获得最佳体验。
  • LESS PRECISION ON COUNTERS* — 开启后，游戏内计数器的小数位将变少。调至 ENABLE 以获得最佳体验。
  • NO BACKGROUND IN MENUS* — 开启后，主菜单将不显示背景。 调至 ENABLE 以获得最佳体验。
  • ALWAYS SIMPLIFY MULTIPLAYER THUMBNAILS* — 开启后， 其它玩家的场地将以较低画质展现。调至 ENABLE 以获得最佳体验。
  • SHOW DUELS SIDE-BY-SIDE* — 开启后，单挑将并排显示。 调至 DISABLE 以获得最佳体验。
  • HIDE CHAT WHEN INGAME* — 开启后，游戏内将不显示聊天框。调至 ENABLE 以获得最佳体验。
  • WARN ME WHEN I'M IN DANGER* — 开启后，若方块堆较高，框体将变红。 调至 DISABLE 以获得最佳体验。
  • STEREO* — 控制立体声效果。 调至 0% (OFF) 以获得最佳体验。
  • DISABLE SOUND ENTIRELY — 开启后，游戏不会播放音乐或音效。游戏会更流畅，但缺点也显而易见。调至 ENABLE 以获得最佳体验。

6. 关闭无关程序和/或浏览器标签

不要在挂着四款游戏的同时游玩 TETR.IO。本游戏是独立的客户端游戏，而不是简单的浏览器标签。因此，请给游戏留下足够多的运行资源，好让他充分利用电脑性能。

7. 将游戏窗口拉小，或调低分辨率

若将游戏窗口拉小，或调低显示器分辨率，游戏需要的算力将降低。这将显著提升游戏的流畅度。

这些我都试过了！！！

如果试过了以上所有方式，TETR.IO 还是卡得不行，那么你的电脑配置似乎不足以游玩本游戏。如果需要进一步的帮助，请通过 Windows+R 打开“运行”，输入 "dxdiag"，将所有数据保存到 .txt 文本文档中。附上这份文档，向社区成员或是 osk 本人反馈。

手柄controllers

TETR.IO 全面支持手柄操作，包括不同动作的手柄震动。不管你使用的是 TETR.IO 桌面版还是现代浏览器，你都可以使用手柄流畅游玩 TETR.IO，只需将其连上电脑即可。TETR.IO 能够自动识别 Xbox、Switch Pro 手柄，并为它们自动设置键位。对于 PlayStation 手柄，可在 Guideline 或 WASD 两套预设键位中选择。若成功连接，每次启动游戏并按下手柄按键时，右下角都会弹出手柄连接的提示。

若手柄未能正确识别，或使用自定义按键设置，可在游戏内手动设置键位：

若手柄无法正常连接，可尝试注意以下问题：

  • 首先，确保未启用有键位映射功能的软件，包括 Steam、JoyToKey；
  • 确保你使用的是唯一连接至电脑的控制器。TETR.IO 只接受来自“端口 1”的控制器（可通过其他方法解决这一问题）。若已连接其他控制器，在断开后需要重启电脑才能生效。

若以上两个方法均未能解决问题，以下是两个针对不同手柄的软件：

  • PS4/PS5's DualShock 系列：使用 DS4Windows
  • Nintendo Switch's Joy-Con 或 Pro 手柄：使用 BetterJoy 的 "ShowAsXInput" function

TETR.IO Desktop Specific Troubleshooting

指定图形处理器dgpu

若感觉笔记本电脑中 TETR.IO 网页版或桌面版游戏很卡，请确保 Windows 使用的是“独立图形处理器”，而不是牺牲画质换取功效的“集成图形处理器”。下面给出一张为游戏指定图形处理器的大图：

在Windows 11 中：
  1. 于开始菜单打开设置（或使用 Win + I 快捷键）；
  2. 搜索“屏幕”设置项，点击“相关设置”中的“显示卡”设置项；
  3. 选择“添加桌面应用”。点击浏览，在弹出的窗口中输入 TETR.IO 桌面版或浏览器的启动路径；
  4. 选中桌面版或浏览器的 .exe 文件；
  5. 选中 TETR.IO 或浏览器。点击选项，再点击“高性能”，完成。
  • TETR.IO 桌面版 .exe 文件的路径：%localappdata%\\Programs\\tetrio-desktop\\resources
  • Chrome.exe 的路径：%programfiles%\\Google\\Chrome\\Application\\chrome.exe、 %programfiles(x86)%\\Google\\Chrome\\Application\\chrome.exe、或 %localappdata%\\Google\\Chrome\\Application\\chrome.exe
  • Firefox.exe 的路径： %programfiles(x86)%\\Mozilla Firefox\\ or %programfiles%\\Mozilla Firefox\\后两者的路径根据你使用 32 位系统或 64 位系统决定。

尝试在 macOS 上安装 TETR.IO 时，系统会弹出信息，警告程序的开发者未认证。若从正规渠道，也即https://tetr.io/about/desktop官方下载链接获取安装程序，可忽略该警告。要想安装，只需右击或control - 单击安装文件，在弹出的窗口中选择“打开”（Open）即可。

若仍无法安装，请参阅苹果官网给出的这篇指南。

安装路径installpaths

使用 TETR.IO 桌面版会在以下路径创建文件夹，使用 Windows + R，在弹出的窗口复制以下路径：

  • %localappdata%\Programs\tetrio-desktop （含有 .exe 文件与 Resources 文件夹）
  • %localappdata%\tetrio-desktop-updater （你所下载的 TETR.IO 桌面版对应版本的安装程序，运行可抹去你对游戏做出的任何修改）
  • %appdata%\TETR.IO （有一名为 “logs” 的空文件夹）
  • %appdata%\tetrio-desktop （包含 Electron 相关的浏览器文件，如 Cache 与 Local Storage）

回放录像

加载回放的方式很简单，但很多玩家不知道如何操作。本章节将解决这一问题。参见：

  • Tenchi 的 FAQ 中的这些条目：
    • Where replays are saved
    • When replays are saved
    • Visual language within the replay timelines

要观看本地保存的录像，只需将资源管理器中的录像文件拖拽至 TETR.IO 窗口中即可。若游戏无法加载录像，请保证回放命名正确，尤其是以下几点：

  • 单人游戏的回放使用 .ttr 后缀；
  • 多人游戏的回放使用 .ttrm 后缀；
  • 自定义模式的预设使用 .ttp 后缀；
  • 设置配置文件使用 .ttc 后缀。

若加载 .ttrm 文件，点击任意一局以查看那一局的回放。

私房回放ttrm

若跳过了结算页，未保存回放，可在聊天框输入 /savereplay 指令保存上一场游戏的录像。

私房回放的要求较为严苛，所以，就算没有“下载回放”的按钮，也不要怪自己。具体而言，只有满足以下所有条件的对局才会生成回放：

  • 必须是一对一的对局，也即参与游戏的玩家必须只有两个；
  • 你的客户端必须启用 LOW 或更高的画质；
  • 若使用 MINIMAL 画质，则必须开启 "SHOW DUELS SIDE BY SIDE" 选项；
  • 必须打完一整场比赛（比如 FT5），从比赛的开始到结束；
  • 当然，还必须在结算页面下载回放文件，下载按钮位于第二名得分条的右下角。

移动设备支持

首先需要澄清一下谣言，TETR.IO 永远不会正式支持移动设备。

话虽如此，玩家依然可以在移动设备上游玩 TETR.IO。请注意：iOS 设备的兼容性有待测试。欢迎玩家使用其他安卓设备测试游玩，但由于移动设备不受官方支持，游戏的兼容性不受保障。

要在手机登移动设备上游玩，只需通过浏览器访问 https://tetr.io/ 即可。游玩的最大问题在于如何控制，你可以使用蓝牙或 USB 连接输入设备，在配置中设置键位。

或者，使用 JavaScript 代码制成的书签来支持屏幕点按操作。可访问这个网站获取更多帮助。
