账户设置

像其他在线服务一样，TETR.IO 的用户通过账户与他人区分。注册用户 可通过多种免费或付费的方式自定义游戏形象。下面是一些有关账户创建或自定义的问题。

注册register

欢迎游玩 TETR.IO！虽然可以简单地注册一个 "匿名" 账户，但只有正式注册的账户才能游玩 TETRA LEAGUE、保存 40 行与闪电战的记录、自定义个人信息与用户页（例如个人头像）等功能。

要创建 TETR.IO 账户，首先https://tetr.io/。若未注册，游戏会自动弹出注册对话框，然后：

  1. 输入用户名（1.1），点击 JOIN 按钮（1.2）
  2. 若提示你输入密码，则说明该用户名已被占用（2）：此时，需要退出并重新起一个用户名；
  3. 反之，若用户名未被占用，游戏会询问你保持匿名（STAY ANONYMOUS）或注册账户（REGISTER）（3）；
  4. 点击 REGISTER 按钮后，游戏会要求你输入邮箱与密码，其中邮箱为推荐项。

注意：根据https://tetr.io/about/rules/给出的小号政策，一名玩家只能拥有一个账户。请仔细检查账户名称：注册后唯一更改的方式是购买 TETR.IO 支持者。

登陆账户login

登陆账户并不困难。游戏的注册页面与登陆页面是一体的，要登陆已注册的账户，请按以下步骤操作：

  1. 游戏提示输入用户名以加入（ENTER A USERNAME TO JOIN）时（1），输入已注册账户的用户名，然后点击 JOIN（1.2）；
  2. 此时（2），游戏提示输入密码。输入密码后（2.1），点击 JOIN（2.2），这样就登陆游戏了。

重置密码passreset

若忘记密码，点击 I FORGOT（2.3）。此时游戏要求你提供注册账户时填写的 Email 地址。请确保填写的地址与注册时填写的一致。游戏若提示错误信息，请多尝试，可能只是输错了地址中的几个字。

注册匿名账户registeranon

TETR.IO 设计之初就想和其他 .IO 类游戏一样上手既玩，因此额外设计了虽有些许功能限制的匿名账户。此外，匿名账户不受小号政策影响，比如，如果想使用不同的输入设备游玩，就可以注册一个匿名账户来避免影响主账户的成绩。要注册匿名账户，只需点击 JOIN 按钮，游戏会自动为你生成一个用户名。匿名账户也可自创用户名，前提是该用户名不与现有的注册用户重复：

  1. 未登录 https://tetr.io/ 时（1），在窗口中输入用户名（1.1），然后点击 JOIN（1.2）；
  2. 若输入的用户名已被占用（2），游戏会提示输入密码。此时需要返回重新选择用户名（2.1）；
  3. 若该名字未被占用，点击保持匿名（STAY ANONYMOUS）即可注册匿名账户。




用户头像pfp

注册的 TETR.IO 用户可通过账户设置免费更改头像。下图展示了该设置项的具体位置，你可以在游戏中的 CONFIG 选项中找到它。

我的头像/横幅是全年龄！notnsfw

TETR.IO 回自动检查用户上传的头像与横幅，其依据是社区规则的第八条。该系统通过由机器学习构建的开源库运作，具体而言，是 GitHub 上的 NSFWJS 与 TensorFlow这两个项目，它们都已列在游戏的开源项目借用表。

注意：NSFWJS 网页上使用的版本可能比 TETR.IO 实际使用的版本更高。

因此，检测系统可能会误判图片，并因此拒绝用户上传。用户有办法回避这一检查机制，但公布这一方法对游戏有较大风险，因此最直接的方式是换一张图片。

举报 TETR.IO 用户report

玩家信息卡的局部放大图。“举报用户”按钮位于“更多选项”选单中。

若想要举报违反the TETR.IO 社区规则的用户，请使用游戏内的举报系统。管理组不接受其它平台的举报（如 Discord 频道）。

只有注册用户才能提交举报，要举报另一名用户，可点击其资料卡片或用户名，或是在游戏内四连频道 (TETRA CHANNEL) 的搜索框中输入该用户名。资料卡中，点击箭头，然后点击举报图标以进入举报流程。在房间中提交的举报将自动附上该房间的聊天记录。

Supporter 等级supporter

TETR.IO Supporter（TETR.IO 支持者）的等级由捐赠的总额度确定 —— 包括个人充值、赠予他人、与旧 Supporter 系统中的 Patreon 支持。可前往 CONFIG > ACCOUNT 查看。

每个等级的最低捐献额度如下：

  • €0   | 一级
  • €20  | ★ （二级）
  • €80  | ★★ （三级）
  • €250 | ★★★ （四级）

达到新的支持者等级后，该等级将永久保留。若支持者身份过期，徽章和昵称颜色暂时消失，但可在重获支持者身份后重新显示。 若捐赠总额度达到 200 欧元，则你的昵称会永久登入制作者名单。即使支持者身份过期，名单也不会将你的昵称移除。

认证verified

TETR.IO 的身份认证的授予由多种原因 —— 大体上有以下几种：TETR.IO 制作组成员（如 TETR.IO 开发者 OSK）、在 TETR.IO Discord 服务器的 Contributer 组的用户（如TETR.IO PLUS 的作者 UNIQMG）、取得杰出成就的玩家（如40行前世界纪录保持者 VINCEHD）、对社区有突出贡献的玩家（如 Underdogs Cups 系列锦标赛的主办方 CABOOZLED_PIE）。

此外，游戏在少数情况下也会将身份认证授予 TETR.IO 社区之外的知名人士（如osu! 知名主播 BTMC）。这类身份认证和一般的身份认证一样，都需通过具体情况分析授予。

自定义项目

玩家可在 TETR.IO 中自定义背景与方块皮肤。

游戏背景custombg

也可使用其他图床网站，如 Imgur，但 Zaptor 还是推荐近几年来因为简单方便而流行的 catbox.moe。

问：怎么自定义背景？

答： 假设背景图片位于你的设备上，此时需要将该图片上传至catbox.moe。该网站免费提供图床服务，且恰好符合 TETR.IO 的 CORS header 需求。

故障排查

若已把图片上传到互联网（包括Catbox），但游戏未能正确显示你所给连接的图像，请通过该网站检测你的图片是否可用。

若可用，请通过 CTRL+F5 来强制刷新游戏（桌面版也需要）。刷新后，随意打开一个模式（如40行），确认图片是否正常加载。

下图英文：确保背景可见度（BACKGROUND VISIBILITY）调高，并且勾选使用自定义背景（USE CUSTOM BACKGROUND IMAGE）选项。

自定义皮肤customskins

安装第三方插件，即代表您放弃报告 TETR.IO 错误的权利！ 若报告问题时安装了 TETR.IO PLUS 等第三方插件，该问题报告无效，并将被清除，即使问题不由第三方插件引起。反复提交这类问题，游戏管理组有权限制您在 Discord 频道与 GitHub 仓库的权限。

TETR.IO 本身不支持自定义皮肤，但可通过第三方插件来手动加载皮肤。其中一款插件叫做 TETR.IO PLUS，详见该开源程序的 GitLab Wiki。

安装好第三方插件后，可前往这些收录了适合 TETR.IO PLUS 使用的皮肤网站。（这些网站并不隶属于 TETR.IO）：

  • you.have.fail/ed/at/tetrioplus 皮肤仓库
  • Tenchi 的皮肤展览
  • Jstris 自定义皮肤库

若对游戏后期的自定义系统有任何建议，请前往 TETR.IO GitHub 仓库下的这个置顶 Issue 并留下评论。

游戏内素材

本章节将给出 TETR.IO 使用的素材，供玩家浏览并下载。

Musicmusic

目前，TETR.IO 的音乐由无版权音乐组织 HURT RECORD 提供，乐曲由多名音乐人创作。任何人都可自由传播这些音乐，只需标注其曲名及来源即可。详见HURT RECORD 的服务条款。

EricICS 使用 Google Docs 制作了音乐列表，并将游戏中使用的音乐以视频的形式上传到了YouTube。xb2002b 则将后者搬运到了 BiliBili。

字体fonts

Authored by osk, here's a small sheet illustrating TETR.IO's fonts.

TETR.IO 使用的字体如右图所示，下面给出它们的下载地址，下载完毕后在资源管理器中右键点击安装即可。

  • HUN-din：（游戏的主字体，全大写，大写字母加宽）https://tetr.io/res/font/hun2.ttf?v=9
  • ProFontWindows:（聊天、房间 ID、回放 ID 等 HUN-din 不适用时采用的替代字体）https://tetr.io/res/font/pfw.ttf
  • Config: （关于页、管理页使用的字体）https://tetr.io/res/font/cr.ttf
  • Franklin Gothic：（osk 徽标使用的字体）为 Windows 自带字体，游戏中使用的是 Franklin Gothic Medium。

音效sound-effects

TETR.IO 的所有音效都存放在名为“音效集”（sound effect atlas）的文件中，可在这里访问。该文件有些问题：所有音效按照时间排列，而不是文件名字。这可能不是你想要的那份文件。如果想要下载含有所有单独音效的文件包，请下载该 .zip 文件（音质较不如音效集），该文件由UniQMG通过 音效提取器制作。

背景图片backgrounds

TETR.IO 使用的背景图片和背景音乐一样，也是自由版权资源。这些图片来自 Unsplash。你也可以自由地使用该网站上的其他图片来替代游戏中的默认背景。

若想下载 TETR.IO 的默认背景，请浏览这张由 @booyah#6473 创建的 Google Sheets。该表格囊括了 TETR.IO 三个测试阶段（INDEV、INFDEV、ALPHA）的所有背景图片。
